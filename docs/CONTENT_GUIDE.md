# Content Guide

## 物品设计原则

每个物品至少定义：

- 名称
- 价值范围
- 类型：普通旧物、工业件、收藏品、异常品、危险品
- 材质暗示
- 网格形状
- 轮廓候选误导
- 一句生活化备注

## 轮廓误导

同一轮廓应对应多个价值差异明显的物品。

示例：

| 轮廓 | 低价值 | 高价值 |
| --- | --- | --- |
| 1x3 长条 | 生锈管件 | 旧式潜水刀 |
| 2x2 方块 | 坏掉的烤面包机 | 密封异常电池 |
| L 型 3x2 | 破排气管 | 绝版街机控制面板 |

## 异常物语气

异常物应该像二手市场商品一样被描述，而不是像史诗神器。

示例：

- “永远指向昨天的罗盘。适合怀旧，不适合导航。”
- “自鸣收音机。没装电池，但会在天气变坏前播广告。”
- “潮湿的玻璃手。估价师说不要问它为什么有指纹。”

## 角色设计方向

角色由两部分组成：

- 被动倾向：稳定影响玩法节奏。
- 动态技能池：玩家只能选择是否使用，不能指定触发哪一个。

早期角色建议：

- 老码头估价师：更擅长识别工业件和收藏品。
- 旧货节目常客：擅长诈唬，技能情报更戏剧化。
- 异常品保险员：更容易识别危险品。
- 破产打捞员：低现金时有更强的跟价韧性。
